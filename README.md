# FlApp

Flask App for learning flask, react, psql, docker etc.
in some parts based on https://blog.miguelgrinberg.com/post/create-a-react-flask-project-in-2025

## Set dev env for flask
New branch for current work
```
git checkout -b flask_init
```

Create dir
```
mkdir flask_back
cd flask_back
```
create and activate python env
```
python -m venv venv
source venv/bin/activate
```

install flask
```
pip install flask python-dotenv
```

create app.py and start building the app

run (test) flask app
```
flask run
```
will do. 
Other methods:

```
flask --app <app_name> run
python app_name.py
```
